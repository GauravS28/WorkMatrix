# WorkMatrix 🚀

### Where Teams Work Smarter

WorkMatrix is a modern and scalable Employee Management System built using React, Vite, and Tailwind CSS. It helps organizations efficiently manage employees, track tasks, and streamline daily operations through a clean and intuitive interface.

---

## ✨ Key Features

* 📊 Interactive Dashboard with employee insights
* 👨‍💼 Employee Management (Add, Edit, Delete)
* 📋 Task Management System
* 🔍 Search & Filter functionality
* 📱 Fully Responsive UI

---

## 🛠 Tech Stack

* **Frontend:** React.js
* **Styling:** Tailwind CSS
* **Build Tool:** Vite

---

## 📁 Project Structure

```
src/
 ├── components/
 │   ├── Auth/
 │   ├── Dashboard/
 │   ├── TaskList/
 │   └── other/
 ├── context/
 ├── utils/
 └── main.jsx
```

---

## 🚀 Getting Started

Clone the repository and run locally:

```
npm install
npm run dev
```
